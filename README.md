instrument-classify
===================

A project to classify instruments by sound using Fourier analysis and ML algorithms

### Source files

| File        			| Description   					|
| --------------------- | --------------------------------- |
| main.ipynb      		| ML algorithms						|
| fft.ipynb      		| Compute FFT of audio file      	| 
| convert-wav.ipynb 	| convert WAV files to 16 bit		|
